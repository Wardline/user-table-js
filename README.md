# user-table-js

Тестовое задание: React-приложение для отображения таблицы пользователей с сортировкой, фильтрацией, пагинацией и модальным окном подробной информации.

## Описание задачи

Необходимо реализовать таблицу пользователей на **React 18+** с получением данных по HTTP и поддержкой:

- сортировки (3 состояния: `asc` / `desc` / `none`) по полям:
  - ФИО
  - возраст
  - пол
  - номер телефона
- колонок таблицы:
  - фамилия, имя, отчество, возраст, пол, номер телефона, email, страна, город
- ограничений:
  - ширина таблицы `100%`, но не более `1400px`
  - для HTTP используется только **Fetch API**

Источник API: [DummyJSON Users](https://dummyjson.com/docs/users)

---

## Реализованный функционал

### Основной

- Загрузка пользователей через HTTP-запросы к DummyJSON
- Таблица пользователей с нужными колонками
- Сортировка по ТЗ с тремя состояниями (`asc` / `desc` / `none`)
- Адаптивная ширина таблицы (`100%`, `max-width: 1400px`)
- Использование только Fetch API для запросов

### Дополнительный

- Фильтрация по соответствующим полям
- Постраничный вывод (pagination)
- Модальное окно по клику на строку таблицы:
  - ФИО, возраст, поля адреса, рост, вес, телефон, email, аватар
- Обработка ошибок HTTP-запросов
- Изменение ширины колонок таблицы
  - минимальная ширина столбца: **50px**
